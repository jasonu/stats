When run (executed), the program does the following:

  1. Clears lists 1, 2 and 3.
  2. Prompts you for a number, N, which is the number of coin flips to perform.
  3. Generates N, 0's or 1's (pseudo-randomly) and saves the results in list 1. (0=tails, 1=heads)
  4. Initialize an accumulator variable, A, to 0. A will keep track of the total number of heads.

  5. Set I=1 (I for “index”) then for each coin flip:

    • Add the coin value to A,
    • Store the value of I into list 2,
    • Store the current proportion of heads, A/I into list 3.

    Increment I by 1. Exit the loop if I > N, otherwise repeat the loop.
