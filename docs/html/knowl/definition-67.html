Definition 5.5.14. Permutations with Non Distinct Items.
\begin{equation*} \frac{n!}{n_1! \, n_2!\, \cdots \, n_k!} \end{equation*}
  1. The \(n\) objects are not distinct, with \(k\) different types of objects.
  2. \(n_1\) is the number of the first type, \(n_2\) the number of the second type, etc.
  3. Repetition of objects is not allowed.
  4. Order is important.
