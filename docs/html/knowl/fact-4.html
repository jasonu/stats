Fact 5.4.3. General Multiplication Rule.
\begin{gather*} P(E \text{ and } F) = P(E) \cdot P(F|E) \end{gather*}

Notice that the general multiplication rule follows from the conditional probability rule.

Two events E and F are independent if \(P(E|F) = P(E)\) or, equivalently, if \(P(F|E) = P(F)\text{.}\) This is the same thing as saying that knowing that one event occurred does not affect the probability of the other event occurring (this is the definition of independence from SectionÂ 5.3).
