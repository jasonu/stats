The term distribution is just a fancy word for a function. We can use the histogram we just created to get a frequency distribution. The \(x\) values of the distribution must correspond with the midpoints of each bin in the histogram. The \(y\) values of the frequency distribution are the frequencies or heights of each bar. Using the graph of the histogram and the trace button we can get each midpoint and frequency and record them in a table.
