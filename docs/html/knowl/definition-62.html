Definition 5.1.5.

The sample space, \(S\text{,}\) of a probability experiment is the collection, or set of all possible outcomes.

An event is any collection of outcomes from a probability experiment. Thus an event is a subset of a sample space. It is common to denote events with capital letters such as \(E, F, G, E_1, E_1, \ldots, E_n\) etc..
